Swiggy Sales Dashboard (Excel Project)
Project Overview

This project is an interactive Swiggy Sales Dashboard built using Microsoft Excel.
The dashboard analyzes food delivery sales data to provide insights into sales performance, order patterns, and customer behavior. It helps understand business trends through visual reports and interactive filtering.

Objective

The main objectives of this project are to:

Analyze overall sales performance and order trends

Track key business metrics using KPIs

Visualize category-wise and restaurant-wise performance

Identify customer ordering patterns

Enable interactive filtering for detailed analysis

Tools and Features Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers (Interactive Filters)

Data Cleaning and Aggregation

Dashboard Design and Formatting

Key Metrics (KPIs)

Total Sales

Total Orders

Average Order Value

Top Selling Categories

Revenue by City / Location

Monthly Sales Trend

Dashboard Insights

The dashboard includes:

Sales by Category

Sales by Restaurant

Monthly Sales Trend Analysis

Orders by City

Top Performing Items

Customer Order Distribution

Interactive slicers allow filtering by:

Category

City

Restaurant

Time Period
